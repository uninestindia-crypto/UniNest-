# UniNest First-Time User Experience (FTUE) Test Script

**Purpose:** This script is for someone who has NEVER seen your app before. Give them this document and watch (don't help!) as they complete the tasks.

---

## For the Tester

Thank you for helping test UniNest! Please complete the tasks below **without any help**. If you get stuck or confused, that's valuable information—please note it.

### Before You Start
- Use **Google Chrome** (latest version)
- Clear your cookies first, or use an **incognito window**
- Turn on **screen recording** if possible (Loom, OBS, or built-in screen recorder)
- Set a timer: the entire test should take **10-15 minutes maximum**

---

## For the UniNest Team

### Scoring the Test

| Score | Interpretation |
|-------|----------------|
| **All tasks completed, no confusion** | Ready to launch |
| **1-2 confusion points** | Minor fixes needed, can launch |
| **3+ confusion points** | Significant UX issues, delay launch |
| **Tester gave up on any task** | Critical UX failure, must fix |

### Key Metrics to Track

- **Time to complete Task 2 (Signup):** _____ minutes
  - Target: < 2 minutes
  - If > 5 minutes: FAIL

- **Time to complete Task 4 (Main feature):** _____ minutes
  - Target: < 3 minutes
  - If > 5 minutes: FAIL

- **Number of times tester said "I don't know what to do":** _____
  - Target: 0
  - If > 2: FAIL

- **Did tester complete all tasks without asking for help?**
  - ☐ Yes → PASS
  - ☐ No → Note where they needed help, those are bugs
